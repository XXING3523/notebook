## 字体

##### 在浏览器中检查字体文件是否成功加载（以 Chrome 浏览器为例）

访问你的 MkDocs 生成的页面，或者包含自定义字体 CSS 的网页。

1. **打开开发者工具**：Windows / Linux：按 `F12` 或 `Ctrl + Shift + I`
2. **切换到 “Network”（网络）面板**：开发者工具打开后，顶部标签选择Network。
3. **刷新页面**：在打开Network面板状态下，（F5 或 Ctrl+R）刷新页面，确保加载所有资源。
4. **筛选字体文件**：在过滤栏中输入字体文件部分名称：`ttf`（TrueType 字体文件）、或关键字，如 `KiwiMaru` 或 `ZiTiGuanJiaFangSongTi`
5. **查看请求状态**：查找到两个字体文件的请求条目，关注Status列：若状态码是 **`200`** （OK），表示字体文件加载成功，如果是 **`404`** （Not Found）或者其它错误码，说明字体文件路径或名称错误，未能加载成功。
