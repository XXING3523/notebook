    P2285 打鼹鼠

    在 n \times n 的网格中，在某些时刻鼹鼠会在某一个网格探出头透气。

    你可控制一个机器人打鼹鼠，如果 i 时刻鼹鼠在某网格中出现，而机器人也处于同一网格，则此鼹鼠可被杀死。

    机器人每一时刻只能移动一格或停在原地不动。移动指从当前所处网格移向相邻网格，即 (i, j) 网格移向 (i-1, j), (i+1, j), (i, j-1), (i, j+1) 四个网格，机器人不能走出整个 n \times n 网格。

    你可自由选定机器人的初始位置。

    现在知道一段时间内鼹鼠出现的时间和地点，求杀死鼹鼠最大值。

    输入格式

    第一行为 n, m\ (1 \le n \le 10^3, 1 \le m \le 10^4)，其中 m 表示这一段时间内出现鼹鼠个数。

    接下来 m 行中每行有三个数据 t, x, y\ (1 \le t \le 2.5 \times 10^7, 1 \le x,y \le n)，表示游戏开始后 t 个时刻，在第 x 行第 y 个网格里出现了一只鼹鼠。

    t 按不降顺序给出。

    注意：同一时刻可能出现多只鼹鼠，但同一时刻同一地点只可能出现一只鼹鼠。

    输出格式

    仅包含一个非负整数，表示被打死鼹鼠的最大数目。

    输入输出样例

    输入

    2 2          
1 1 1       
2 2 2

    输出

    1
